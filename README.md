## About

This repository contains my exercises, guided projects, and challenge projects completed while following the **Microsoft Learn x freeCodeCamp** learning program for C# (the **Foundational C#** curriculum / certification preparation).

Microsoft and freeCodeCamp partnered to provide a **free, globally accessible Foundational C# Certification**, backed by a Microsoft Learn training course. ([devblogs.microsoft.com](https://devblogs.microsoft.com/dotnet/announcing-foundational-csharp-certification/?utm_source=openai))

## What I learned (high level)

The course focuses on building solid C# fundamentals, especially for **console applications**, including:

- Setting up a C# dev environment (commonly Visual Studio Code + .NET tooling)
- C# syntax, variables, and basic operations
- Decision logic (`if / else if / else`, `switch`)
- Iteration (`foreach`, `for`, arrays)
- Methods (parameters, return values, optional/named arguments)
- Debugging console apps

Microsoft Learn also provides learning paths such as “Create methods in C# console applications”, including guided and challenge projects. ([learn.microsoft.com](https://learn.microsoft.com/en-us/training/paths/get-started-c-sharp-part-5/?utm_source=openai))

This repo is organized by course parts/sections:

## Getting started (run locally)

### Prerequisites

- Install the **.NET SDK** (version depends on the projects; use a recent stable release).
- Any of:
  - **Visual Studio Code** (recommended)
  - **Visual Studio**
  - CLI only

### Run with the .NET CLI

From a project folder that contains a `.csproj` file:

```bash
dotnet run
```

### Open the solution

Open `C#_Learning.sln` with Visual Studio, or open the repo in VS Code and run/debug from there.
